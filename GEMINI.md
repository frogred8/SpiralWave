# SpiralWave 프로젝트 개요

## 게임 설명
SpiralWave는 `idle healing game`입니다. 플레이어는 화면 중앙에 있는 신비로운 'Spiral'의 관리자가 되어, 우주 공간을 떠다니는 다양한 에너지 입자들을 끌어당기고 흡수합니다. 게임이 진행됨에 따라 플레이어는 Spiral의 **Radius**를 업그레이드하여 더 넓은 영역을 제어하고, **Force**를 강화하여 더 멀리 있는 입자들을 끌어당길 수 있습니다. 입자들이 Spiral로 빨려 들어가는 부드러운 애니메이션은 시각적 만족감을 줍니다. 인터페이스는 직관적이며, 복잡한 입력 없이 마우스나 터치로 제어됩니다. 또한, 마우스 클릭으로 로봇팔을 발사해 멀리 있는 자원을 직접 가져오는 능동적인 플레이도 가능합니다. 레벨이 올라가면 새로운 종류와 패턴의 입자들이 나타나므로, 성장을 최적화하기 위해 전략적인 업그레이드 계획이 필요합니다.

### 세부 기획
-   **자원 종류:** 돌, 나무
-   **획득 방식:**
    -   **패시브 (Spiral 중력):** 입자들이 Spiral 근처로 들어오면 중력에 의해 점차 그 중심부로 이끌려 결국 흡수됩니다.
    -   **패시브 (그물):** 일정 시간마다 중심부에서 마우스 커서 방향으로 그물을 던져 범위 안의 자원을 회수합니다.
    -   **액티브 (로봇팔):** 플레이어가 마우스를 클릭하면, 클릭한 지점으로 로봇팔이 쭉 늘어나서 자원을 붙잡아 빠르게 회수합니다.

## 기술 스택
-   **게임 엔진:** Phaser 3 (Phaser.js)
-   **언어:** TypeScript
-   **실시간 통신:** Socket.io
-   **모듈 로딩:** Blob URL + Dynamic Import

## 현재 작업: RPG 스킬 트리 구현

스킬 트리를 세 개의 tree와 네 개의 row로 구성합니다. 구현은 다음 핵심 요구사항을 따라야 합니다:

### 구조
-   각 스킬은 **grid** 레이아웃으로 배열되어야 합니다.
-   스킬은 **node-based dependency** 구조를 따라야 합니다: 상위 스킬을 배워야 하위 스킬이 해금됩니다.
-   스킬 간의 의존성을 보여주는 **시각적인 연결선**(수직/대각선)이 필요합니다.
-   스킬은 게임 시작할 때 무작위 배치됩니다.
-   선행 스킬은 자신의 바로 윗칸의 스킬과 각 row마다 이웃하는 트리의 상위 스킬을 연결합니다.
  -   단, 마지막 row는 선행 스킬을 하나 더 가집니다.

### 상호작용
-   **스킬 레벨 업:** 스킬 아이콘을 클릭하면 스킬 레벨이 올라가며, 스킬당 개별적으로 최대 레벨이 지정되어 있습니다.
-   **툴팁 표시:** 마우스를 올리면(`mouse-over`), 스킬의 이름과 현재 레벨/효과를 보여주는 **tooltip**이 나타납니다.
-   **시각적 잠금 해제 상태:** 아직 잠금 해제되지 않은 스킬은 **grayscale**로 표시되어야 합니다. 잠금이 해제되면 **color**로 변경되어야 합니다.
-   자원 획득 시 자원 표시하는 UI와 중심부에 획득한 자원이 +N으로 잠깐 표시됩니다.

### 데이터 구조
-   **JSON 관리:** 스킬 데이터(이름, 값 등)는 향후 쉽게 수정할 수 있도록 **JSON format**으로 관리되어야 합니다.
-   JSON 데이터는 관련있는 필드만 최소한으로 수정합니다.

### 시각적 스타일
-   **UI** 버튼은 **어둡고 거친 금속 느낌**이어야 합니다.
-   배경은 **돌 질감**으로 합니다.
