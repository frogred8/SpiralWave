# 스킬 트리 데이터 테이블

이 문서는 게임 내 스킬 트리의 데이터 구조를 정의합니다. 각 스킬 트리는 탭으로 구분되며, 스킬들은 테이블 형식으로 관리됩니다.

## 주요 컬럼 설명

-   **ID**: 스킬의 고유 식별자입니다.
-   **Name**: 스킬의 이름입니다.
-   **Description**: 스킬의 설명입니다. `{value}`와 같은 플레이스홀더를 사용하여 동적인 값을 표현할 수 있습니다.
-   **Max Level**: 스킬이 도달할 수 있는 최대 레벨입니다.
-   **Prerequisites**: 스킬을 배우기 위해 필요한 선행 스킬 목록입니다. `ID@레벨` 형식으로 작성하며, 여러 조건이 필요할 경우 쉼표(`,`)로 구분합니다 (예: `C1@5,C2@1`).
-   **Grid Position**: UI 상에 표시될 스킬의 격자 좌표입니다. `(행, 열)` 형식입니다.
-   **Effects per Level**: 레벨이 오를 때마다 스킬 효과가 어떻게 변하는지 정의합니다. JSON과 유사한 `key: value` 형식으로 작성합니다.
